# PhishHunterX - Hybrid Machine Learning Based Phishing Detection System

## Overview
PhishHunterX is a cybersecurity web application developed using Django and Machine Learning to detect phishing websites based on URL analysis.

## Features
- URL-based phishing detection
- Machine learning classification
- User authentication
- Security analysis dashboard
- Phishing prediction results

## Technologies Used
- Python
- Django
- Scikit-Learn
- Pandas
- NumPy
- MySQL

## Project Structure
- Remote_User Module
- Service_Provider Module
- URL Classification Engine
- Dataset Training Module

## Machine Learning Algorithms
- Decision Tree
- Random Forest
- Naive Bayes
- Gradient Boosting

## Dataset
- Datasets.csv
- labeled_data.csv

## How to Run the Project

1. Activate virtual environment
    venv\Scripts\activate

2. Navigate to the project directory
   cd phishing_detection_system

3. Start the Django server
   py manage.py runserver

4. Open the application
   http://127.0.0.1:8000
